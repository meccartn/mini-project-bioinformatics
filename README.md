# 💻 Python Programming & Computational Biology Portfolio

## 📌 Description
This repository serves as a digital portfolio containing a collection of Python scripts, fundamental computational modules, and programming projects. The primary focus of this repository is applying programming logic to solve quantitative data problems and process molecular biology data (Bioinformatics).

---

## 📁 Project Roadmap & Table of Contents

### 🟢 LEVEL 1: Basic Python & Core Fundamentals
This section contains starter projects focused on fundamental programming logic, text manipulation, and basic biology scripting.

#### 1. 🧬 DNA to RNA Transcription
*   **Folder:** `01-DNA-to-RNA-Transcription/`
*   **Description:** Built an automated DNA-to-RNA transcription program utilizing string replacement methods and linear sequence validation (`if all()`) for molecular biology modeling.
*   **Status:** Completed

#### 2. 🔍 DNA Sequence Validation
*   **Folder:** `02-DNA-Sequence-Validation/`
*   **Description:** Developed a DNA sequence validation script using looping algorithms and early return patterns to detect nitrogenous base mismatches {A, T, G, C} for genetic mutation analysis.
*   **Status:** Completed

#### 3. 🛑 AUG Start Codon Detector
*   **Folder:** `03-AUG-Start-Codon-Detector/`
*   **Description:** Designed an mRNA initiation codon detector system based on text manipulation functions (`.startswith()`) to validate the early phase of the RNA translation process.
*   **Status:** Completed
